## Full Course Topics

### Properties and Methods of a Class
- OOP concept explained in simple words
- Classes and objects. Class attributes and instance attributes
- Class methods. The `self` parameter
- Initializer `__init__` and finalizer `__del__`
- Magic method `__new__`. Singleton pattern example
- Class methods (`classmethod`) and static methods (`staticmethod`)
- Access modes: `public`, `private`, `protected`. Setters and getters
- Magic methods `__setattr__`, `__getattribute__`, `__getattr__` and `__delattr__`
- Monostate pattern
- Properties with `property`. The `@property` decorator
- Practical examples of `property` objects
- Descriptors: data descriptor and non-data descriptor
- Magic methods overview
- `__call__` magic method. Functors and class-based decorators
- `__str__`, `__repr__`, `__len__`, `__abs__`
- `__add__`, `__sub__`, `__mul__`, `__truediv__`
- Comparison methods: `__eq__`, `__ne__`, `__lt__`, `__gt__` and others
- `__eq__` and `__hash__` together
- `__bool__` — defining truthiness of objects
- `__getitem__`, `__setitem__` and `__delitem__`
- `__iter__` and `__next__`

### Inheritance and Polymorphism
- Inheritance in object-oriented programming
- `issubclass()`. Inheriting from built-in types and from `object`
- `super()` function and delegation
- `private` and `protected` attributes in inheritance
- Polymorphism and abstract methods
- Multiple inheritance
- `__slots__` collection
- How `__slots__` works with `property` and in inheritance

### Exceptions and Context Managers
- Introduction to exception handling. `try / except` blocks
- `finally` and `else` blocks
- Exception propagation
- `raise` statement and custom exceptions
- Context managers. The `with` operator

### Metaclasses, Nested Classes, Data Classes
- Nested classes
- Metaclasses. The `type` object
- Custom metaclasses. The `metaclass` parameter
- Metaclasses in Django ORM API
- Introduction to Python Data Classes (part 1)
- Introduction to Python Data Classes (part 2)
- Python Data Classes with inheritance
